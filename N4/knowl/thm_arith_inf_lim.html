Teorema 1.12. Aritmética de limites infinitos.

Sejam \(a,c,H \) constantes e \(f,g,h\) funções definidas em um intervalo que contém \(a\) (mas não precisam ser definidas em \(x=a\)), de modo que

\begin{align*} \lim_{x \to a} f(x) \amp= +\infty \amp \lim_{x \to a} g(x) \amp= +\infty \amp \lim_{x \to a} h(x) \amp= H \end{align*}

  • \(\displaystyle \displaystyle \lim_{x \to a} ( f(x) + g(x) ) = +\infty\)

  • \(\displaystyle \displaystyle \lim_{x \to a} ( f(x) + h(x) ) = +\infty\)

  • \(\displaystyle \lim_{x \to a} ( f(x) - g(x) )\) indeterminado

  • \(\displaystyle \displaystyle \lim_{x \to a} ( f(x) - h(x) ) = +\infty\)

  • \(\displaystyle \displaystyle \lim_{x \to a} c f(x) = \begin{cases} +\infty \amp c \gt 0 \\ 0 \amp c=0 \\ -\infty \amp c \lt 0 \end{cases}\)

  • \(\displaystyle \lim_{x \to a} ( f(x) \cdot g(x) ) = +\infty\text{.}\)

  • \(\displaystyle \displaystyle \lim_{x \to a} f(x) h(x) = \begin{cases} +\infty \amp H \gt 0 \\ -\infty \amp H \lt 0\\ \text{undetermined} \amp H=0 \end{cases}\)

  • \(\displaystyle \lim_{x \to a} \frac{f(x)}{g(x)}\)indeterminado

  • \(\displaystyle \displaystyle \lim_{x \to a} \frac{f(x)}{h(x)} = \begin{cases} +\infty \amp H \gt 0 \\ -\infty \amp H \lt 0\\ \text{undetermined} \amp H=0 \end{cases}\)

  • \(\displaystyle \displaystyle \lim_{x \to a} \frac{h(x)}{f(x)} = 0\)

  • \(\displaystyle \displaystyle \lim_{x \to a} f(x)^p = \begin{cases} +\infty \amp p \gt 0 \\ 0 \amp p \lt 0\\ 1 \amp p=0 \end{cases}\)
