Exemplo 1.16. .

Encontre \(y''\) se \(y=y^3+xy+x^3\text{.}\)

Solução Este problema diz respeito a alguma função \(y(x)\) que não é nos foi dado explicitamente. Tudo o que nos é dito é que \(y(x)\) satisfaz

\begin{equation*} y(x)=y(x)^3+xy(x)+x^3 \tag{E1} \end{equation*}

para todos os \(x\text{.}\) Somos solicitados a encontrar \(y''(x)\text{.}\) Não podemos resolver isso equação para obter uma fórmula explícita para \(y(x)\text{.}\) Então usamos implícito diferenciação, como fizemos no Exemplo [cross-reference to target(s) "eg_DIFFimpldiffD" missing or not unique]. Ou seja, aplicamos \(\dfrac{\dd }{\dd x}\) para ambos os lados de (E1). Isto dá

\begin{equation*} y'(x)=3y(x)^2\,y'(x)+y(x)+x\,y'(x)+3x^2 \tag{E2} \end{equation*}

que podemos resolver para \(y'(x)\text{,}\) movendo todos os \(y'(x)\) para a esquerda lado, dando

\begin{equation*} \big[1-x-3y(x)^2\grande]y'(x) = y(x)+3x^2 \end{equation*}

e depois dividindo.

\begin{equation*} y'(x) = \frac{y(x)+3x^2}{1-x-3y(x)^2} \tag{E3} \end{equation*}

Para obter \(y''(x)\text{,}\) temos duas opções.

Método 1. Aplique \(\dfrac{\dd }{\dd x}\) em ambos os lados de (E2). Isto dá

\begin{equation*} y''(x)=3y(x)^2\,y''(x)+6y(x)\,y'(x)^2+2y'(x)+x\,y''(x )+6x \end{equation*}

Agora podemos resolver para \(y''(x)\text{,}\) dando

\begin{equation*} y''(x) = \frac{6x+2y'(x)+6y(x)y'(x)^2}{1-x-3y(x)^2} \tag{E4} \end{equation*}

Então podemos substituir em (E3), dando

\begin{align*} \amp y''(x) = 2\frac{3x+ \frac{y(x)+3x^2}{1-x-3y(x)^2} +3y(x) \big(\frac{y(x)+3x^2}{1-x-3y(x)^2}\big)^2} {1-x-3y(x)^2}\\ \amp= 2\frac{3x{[1\!-\!x\!-\!3y(x)^2]}^2+ [y(x)\!+\!3x^2][1\ !-\!x\!-\!3y(x)^2] +3y(x) {[y(x)\!+\!3x^2]}^2}{{[1-x-3y(x)^2]}^3} \end{align*}
