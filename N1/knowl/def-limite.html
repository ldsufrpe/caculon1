Definição 1.1.

O limite de uma função \(f(x)\) com \(x\) aproximando-se de \(a\) é um número \(L\) com a seguinte propriedade: podemos tornar os valores de \(f(x)\) arbitrariamente próxima \(L\text{,}\) tomando \(x\) suficientemente próximo de \(a\) (tende para \(a\) por ambos os lados), mas não igual a \(a\text{.}\)
