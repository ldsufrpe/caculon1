Solução 1.8.6.1.

Solução

1. Se \(x\) está próximo \(0\) , então o denominador \(x^2(x+7)\) é um número pequeno negativo. Portanto, o queciente \(\frac{1}{x^2(x+7)}\) é um número grande positivo. Então, \(\displaystyle \lim_{x\to 0} \frac{1}{x^2(x+7)}= \infty\text{.}\)

2. Se \(x\) está próximo de \(3\) mas maior que \(3\text{,}\) então o denominador \(x-3\) é um número positivo pequeno. Portanto, o quociente \(\frac{2}{x-3}\) é um número positivo grande. Então, intuitivamente, temos que \(\displaystyle \lim_{x\to 3^{+}} \frac{2}{x-3}= \infty\text{.}\)

3. Se \(x\) está próximo de \(-7\) mas menor que \(-7\text{,}\) então o denominador \(x^2(x+7)\) é um número pequeno negativo. Portanto, o quociente \(\frac{1}{x^2(x+7)}\) é um número grande negativo. Então, intuitivamente, temos que \(\displaystyle \lim_{x\to -7^{-}} \frac{1}{x^2(x+7)}= -\infty\text{.}\)

4. Se \(x\) está próximo de \(3\) mas é menor que 3 \(3\text{,}\) então o denominador \(x-3\) é um número negativo pequeno. Portanto, o quociente \(\frac{2}{x-3}\) é um número negativo grande. Então, intuitivamente, temos que \(\displaystyle \lim_{x\to 3^{-}} \frac{2}{x-3}= - \infty\text{.}\)
