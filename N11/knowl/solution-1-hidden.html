Considere o limite

\begin{gather*} \lim_{x\to 0} \frac{\sin x}{x} \end{gather*}

Note que

\begin{align*} \lim_{x\to 0} \sin x \amp= 0\\ \lim_{x\to 0} x \amp= 0 \end{align*}

então esta é uma forma \(\frac00\) indeterminada, e sugere que tentemos a Regra de L'Hôpital. Para aplicar a regra devemos primeiro verificar os limites das derivadas.

\begin{align*} f(x)\amp= \sin x \amp f'(x) \amp =\cos x \amp \text{and} \amp\amp f'(0)=1\\ g(x) \amp= x \amp g'(x) \amp = 1 \amp \text{and} \amp\amp g'(0)=1 \end{align*}

Então, pela regra de l'Hôpital

\begin{align*} \lim_{x\to 0} \frac{\sin x}{x} \amp= \frac{f'(0)}{g'(0)} = \frac{1}{1} = 1. \end{align*}
