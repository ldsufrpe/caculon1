Definição 1.1. Primeiras formas indeterminadas.

Seja \(a \in \mathbb{R}\) e seja \(f(x)\) e \(g(x)\) funções. Se

\begin{align*} \lim_{x\to a} f(x) \amp= 0 \amp \text{and } \amp\amp \lim_{x\to a} g(x) \amp= 0 \end{align*}

então o limite

\begin{gather*} \lim_{x\to a} \dfrac{f(x)}{g(x)} \end{gather*}

é chamado forma indeterminada \(\frac{0}{0}\text{.}\)
