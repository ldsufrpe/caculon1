Solução 1.6.9.1.

O limite é da forma \(1^\infty\text{.}\) Seja

\(y= (\cos{x})^{1/x^2}\) e portanto \(\ln y = \displaystyle\frac{1}{x^2}\ln(\cos{x})\text{.}\)

Obtemos:

\begin{equation*} \lim_{x \longrightarrow 0 } \frac{1}{x^2}\ln{(\cos{x})} =\lim_{x \longrightarrow 0 } \frac{\ln{(\cos{x})}}{x^2} \end{equation*}

Este limite é da forma \(\frac{0}{0}\) , então aplicamos a Regra de L'Hopital duas vezes:

\begin{equation*} \lim_{x \longrightarrow 0 } \frac{\ln{(\cos{x})}}{x^2} = \lim_{x \longrightarrow 0 } \frac{\frac{1}{\cos{x}}(-\sin{x})}{x^2} = \lim_{x \longrightarrow 0 } \frac{-\tan{x}}{2x} = \end{equation*}
\begin{equation*} = \lim_{x \longrightarrow 0 } \frac{-\sec^2{x}}{2} = \frac{-1}{2} \end{equation*}

Então \(\ln y = -1/2\) e

\begin{equation*} y= \lim_{x \longrightarrow 0} (\cos{x})^{1/x^2} = e^{-1/2} = \frac{1}{\sqrt{e}} \end{equation*}
