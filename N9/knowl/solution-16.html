Solução 1.6.12.1.

Derivamos implicitamente ambos os lados da equação em relação para \(x\text{.}\)

\begin{equation*} \frac{1}{(1 + x^{6} y^2)} (3 x^{2} y + x^{3} \frac{dy}{dx}) = y^{3} + 3 x y^{2} \frac{dy}{dx}, \end{equation*}

então, simplificando o denominador,

\begin{equation*} 3 x^{2} y + x^{3} \frac{dy}{dx} = (1 + x^{6} y^2) (y^{3} + 3 x y^{2} \frac{dy}{dx}). \end{equation*}

Assim

\begin{equation*} ( x^{3} - (3 x y^{2}) (1 + x^{6} y^2) ) \frac{dy}{dx} = y^{3} (1 + x^{6} y^2) - 3 x^{2} y, \end{equation*}

e finalmente

\begin{equation*} \frac{dy}{dx} = \frac{y^{3} (1 + x^{6} y^2) - 3 x^{2} y}{x^{3} - (3 x y^{2}) (1 + x^{6} y^2)}. \end{equation*}
