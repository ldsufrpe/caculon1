Parágrafo

Considere que a base logarítmica \(e\) — \(\log_e(x)\) é a potência que \(e\) deve ser elevada para fornecer \(x\text{.}\) Ou seja, \(\log_e(x)\) é definido por

\begin{align*} e^{\log_e x}\amp= x \end{align*}

ou seja, — é a inversa da função exponencial com base \(e\text{.}\) Dado que esta escolha da base funciona tão clara e facilmente em relação à derivação, esta base acaba por ser a escolha mais natural do logaritmo. É fácil usar logaritmos de uma base para calcular logaritmos com outra base:

\begin{align*} \log_q x\amp= \frac{\log_e x}{\log_e q} \end{align*}

Portanto, somos (relativamente) livres para escolher uma base que seja conveniente para os nossos propósitos.
