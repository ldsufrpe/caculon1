We can justify adapting the rule to the limits to \(\pm \infty\) via the following reasoning

\begin{align*} \lim_{x\to \infty} \frac{f(x)}{g(x)} \amp= \lim_{y \to 0^+} \frac{ f(1/y) }{ g(1/y) } \amp \text{substitute } x=1/y\\ \amp= \lim_{y \to 0^+} \frac{ -\frac{1}{y^2} f'(1/y) } { -\frac{1}{y^2}g'(1/y)}, \end{align*}

where we have used l'Hôpital's rule (assuming this limit exists) and the fact that \(\dfrac{\dd}{\dd y} f(1/y) = -\frac{1}{y^2} f'(1/y)\) (and similarly for \(g\)). Cleaning this up and substituting \(y=1/x\) gives the required result:

\begin{align*} \lim_{x\to \infty} \frac{f(x)}{g(x)} \amp =\lim_{y \to 0^+} \frac{f'(1/y)} {g'(1/y)} =\lim_{x\to\infty} \frac{f'(x)}{g'(x)}. \end{align*}
