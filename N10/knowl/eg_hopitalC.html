Exemplo 3.6. Double L'Hôpital.

Compute the limit

\begin{gather*} \lim_{x\to 0} \frac{\sin(x^2)}{1-\cos x} \end{gather*}

  • Again we should check

    \begin{align*} \lim_{x\to 0} \sin(x^2) \amp= \sin 0 = 0\\ \lim_{x\to 0}(1-\cos x) \amp= 1-\cos 0 = 0 \end{align*}
    and we have a \(\frac00\) indeterminate form.

  • Let \(f(x) = \sin(x^2)\) and \(g(x)=1-\cos x\) then

    \begin{align*} f'(x) \amp= 2x\cos(x^2) \amp f'(0)\amp=0\\ g'(x) \amp= \sin x \amp g'(0) \amp=0 \end{align*}
    So if we try to apply l'Hôpital's rule naively we will get
    \begin{align*} \lim_{x\to 0} \frac{\sin(x^2)}{1-\cos x} \amp= \frac{f'(0)}{g'(0)} = \frac{0}{0}. \end{align*}
    which is another \(\frac00\) indeterminate form.

  • It appears that we are stuck until we remember that l'Hôpital's rule (as stated in Theorem Teorema 3.2) has a part (b) — now is a good time to reread it.

  • It says that

    \begin{align*} \lim_{x\to 0} \frac{f(x)}{g(x)} \amp= \lim_{x\to 0} \frac{f'(x)}{g'(x)} \end{align*}
    provided this second limit exists. In our case this requires us to compute
    \begin{gather*} \lim_{x \to 0} \frac{2x \cos(x^2)}{\sin(x)} \end{gather*}
    which we can do using l'Hôpital's rule again. Now
    \begin{align*} h(x) \amp= 2x\cos(x^2) \amp h'(x) \amp= 2\cos(x^2) - 4x^2\sin(x^2) \amp h'(0) \amp=2\\ \ell(x) \amp= \sin(x) \amp \ell'(x) \amp= \cos(x) \amp \ell'(0) \amp= 1 \end{align*}
    By l'Hôpital's rule
    \begin{align*} \lim_{x \to 0} \frac{2x \cos(x^2)}{\sin(x)} \amp= \frac{h'(0)}{\ell'(0)} = 2 \end{align*}

  • Thus our original limit is

    \begin{align*} \lim_{x\to 0} \frac{\sin(x^2)}{1-\cos x} \amp=\lim_{x \to 0} \frac{2x \cos(x^2)}{\sin(x)} = 2. \end{align*}

  • We can succinctly summarise the two applications of L'Hôpital's rule in this example by

    \begin{align*} \lim_{x\to 0}\underbrace{\frac{\sin(x^2)}{1-\cos x}}_{\atp {\mathrm{num}\to 0} {\mathrm{den}\to 0}} \amp=\lim_{x\to 0}\underbrace{\frac{2x\cos(x^2)}{\sin x}}_{\atp {\mathrm{num}\to 0} {\mathrm{den}\to 0}} =\lim_{x\to 0}\underbrace{\frac{2\cos(x^2)-4x^2\sin(x^2)}{\cos x}}_{\atp {\mathrm{num}\to 2} {\mathrm{den}\to 1}} =2 \end{align*}
    Here “num” and “den” are used as abbreviations of “numerator” and “denominator” respectively."
