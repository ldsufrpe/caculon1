Corolário 1.17.

Seja \(f(x)\) uma função no intervalo \(a \leq x \leq b\text{.}\) Então para encontrar o máximo e o mínimo global da função

  • Faça uma lista de todos os valores de \(c\text{,}\) com \(a\le c\le b\text{,}\) para os quais

    • \(f'(c)=0\text{,}\) ou

    • \(f'(c)\) não existe, ou

    • \(c=a\) ou \(c=b\text{.}\)

    Isto é — calcule a função em todos os pontos críticos, pontos singulares e os pontos extremos.

  • Avalie \(f(c)\) para cada \(c\) nessa lista. O maior (ou menor) desses valores é o maior (ou menor) valor de \(f(x)\) para \(a\le x\le b\text{.}\)
