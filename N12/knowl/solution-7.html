Solução 1.5.3.1.

Para resolver esse problema, precisaremos escrever uma fórmula para o volume da caixa em termos de uma de suas dimensões e, em seguida, usar derivadas para encontrar as dimensões nas quais a caixa tem volume máximo. Seja \(x\) o comprimento dos lados da base quadrada. Então, se \(h\) é a altura da caixa, o volume é dado por \(x^2h\text{.}\) Precisamos encontrar uma expressão para a altura \(h\) em termos de \(x\text{.}\)

É aqui que usamos nossas informações sobre a quantidade de material usado na construção da caixa. Se a base da caixa tem lados de comprimento \(x\text{,}\) então \(x^2\) centímetros quadrados de material são usados para fazer a base. Portanto, temos apenas \(1200 - x^2\) centímetros quadrados de material para fazer os lados, dos quais existem quatro. Cada um dos lados usa \(hx\) centímetros quadrados de material. Assim, obtemos a fórmula:

\begin{equation*} 1200-x^2 = 4(hx) \Rightarrow h = \frac{1200-x^2}{4x} \end{equation*}

Colocando isso em nossa fórmula para volume, agora podemos escrever \(v(x)\) como:

\begin{equation*} v(x) = x^2\left(\frac{1200-x^2}{4x}\right) = \frac{{1200}x - x^3}{4} \end{equation*}

Agora, tomamos a derivada desta expressão, usando as regras para derivar polinômios, para obter \(v'(x) = \frac{1200}{4} - \frac{3}{4}x^2\text{.}\) Definir isso como 0 nos dará os pontos críticos. Ao resolver, lembre-se de que esta é uma situação do mundo real, então não podemos ter um valor negativo para \(x\) (que é um comprimento).

\begin{equation*} \begin{array}{rcl} v'(x) \amp = \amp 0 \\ \frac{1200}{4} - \frac{3}{4}x^2 \amp = \amp 0 \\ \frac{3}{4}x^2 \amp = \amp \frac{1200}{4} \\ x^2 \amp = \amp \frac{1200}{3} \\ x \amp = \amp \sqrt{\frac{1200}{3}} \approx 20.00 \\ \end{array} \end{equation*}

Agora, substituindo essa largura em nossa fórmula para volume, \(v(x)\text{,}\) obtemoso volume máximo de \(v(20.00) = !{4000\ {\rm cm^{3}}:\%5.2f}\text{.}\)
