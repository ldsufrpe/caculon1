Solução 1.8.11.1.

Solução.

Usando a definição da derivada encontramos que:

\(\displaystyle f'(0) =\lim_{ h \to 0 } \frac{f(0) - f(h)}{h}\)

\(\displaystyle = \lim_{ h \to 0 } h^2 sin \left(\frac{1}{h} \right) \frac{1}{h}\)

\(\displaystyle = \lim_{ h \rightarrow 0 } h \sin(\frac{1}{h})\)

\(\displaystyle = 0\text{,}\) pelo teorema do confronto.
