Solução 1.8.8.1.

\begin{equation*} f'(0) = \lim_{h\rightarrow 0 } \frac{f(0+h)-f(0)}{h} = \lim_{h\rightarrow 0 } \frac{10\!\left(0+h\right)-\left(0+h\right)^{2}-0}{h} = 10 \end{equation*}
\begin{equation*} f'(1) = \lim_{h\rightarrow 1 } \frac{f(1+h)-f(1)}{h} = \lim_{h\rightarrow 0 } \frac{10\!\left(1+h\right)-\left(1+h\right)^{2}-9}{h} = 8 \end{equation*}
\begin{equation*} f'(2) = \lim_{h\rightarrow 2 } \frac{f(2+h)-f(1)}{h} = \lim_{h\rightarrow 0 } \frac{10\!\left(2+h\right)-\left(2+h\right)^{2}-16}{h} = 6 \end{equation*}
\begin{equation*} f'(3) = \lim_{h\rightarrow 3 } \frac{f(3+h)-f(1)}{h} = \lim_{h\rightarrow 0 } \frac{10\!\left(3+h\right)-\left(3+h\right)^{2}-21}{h} = 4 \end{equation*}
Supondo que o padrão continue, temos \(f'(4)=2\) and \(f'(5)=0\text{.}\)
